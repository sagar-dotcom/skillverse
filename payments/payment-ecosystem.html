🎭 Meet The Players

👨 Customer

Initiates payment and authorizes transaction.

🛒 Merchant

Sells products and receives payment.

📱 Payment Gateway

Securely transfers transaction data.

🌐 Card Network

Routes messages between banks.

🏦 Issuer Bank

Checks balance and approves payment.

🏦 Acquirer Bank

Provides payment services to merchant.

💰 Where Does MDR Go?

🏦 Issuer Bank

Receives interchange fee.
🏦 Acquirer Bank

Receives acquiring fee.
🌐 Network

Receives network processing fee.
📱 Gateway

Receives technology/service fee.

⚠ Why Does Money Sometimes Get Debited?

🌐 Network Timeout
📱 Gateway Failure
🏦 Bank Processing Delay
🔄 Reconciliation Pending
⚡ System Downtime
📡 Connectivity Issue

🎯 Interview Corner

What is MDR?

Merchant Discount Rate charged for processing transactions.

Issuer vs Acquirer?

Issuer serves customer, Acquirer serves merchant.

Role of Gateway?

Secure communication between merchant and payment ecosystem.

Role of Visa?

Routes payment messages between banks.

📌 Key Takeaways

✅ Customer starts payment

✅ Merchant receives payment

✅ Gateway securely transfers data

✅ Network routes transaction

✅ Issuer approves payment

✅ Acquirer supports merchant

✅ MDR is shared among ecosystem players
